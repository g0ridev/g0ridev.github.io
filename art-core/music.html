Industrial Rhythms
Strudel | 9/25/28
Layered composition using TR-808 drums, distorted bass, and processed vinyl noise. Explores industrial textures through digital manipulation and euclidean rhythms. Built around a simple chord progression with heavy processing and probabilistic elements. I was experimenting with euclid() function of strudel. Mainly tryign to find good melodies in a quick way.
Play in Strudel
setcpm(60) const drums = s("tr808_bd").slow(2) .lpf(200).crush(8).gain(0.8) .distort(2).crush(10).gain(0.2) const bass = note("c2 ~ ~ eb2 ~ f2 ~ ~").s("gm_synth_bass_1").slow(2) .lpf(250).shape(0.5).gain(0.6) const snare = s("~ ~ ~ tr808_cp").gain(0.7).room(0.3).crush(4).slow(2) drums.stack(snare,bass) const vinyl = s("white").slow(3).gain(0.1) .hpf(4000).crush(6) vinyl.stack(drums,snare,bass) const piano = note("c3 eb3 ~ f3 ~ ~ eb3 ~").s("gm_piano").slow(2).sometimes(x=>x.euclid(3,4)) .lpf(600).crush(8).room(0.3).gain(0.4) const chords = note("<[d3,f3,a3] [g3,bb3,d4] [a3,c4,e4] [d3,f3,a3] >").s("fmpiano").slow(4) .lpf(800).gain(0.3).euclid(3,4) piano.stack(drums,snare,bass,vinyl,chords)
